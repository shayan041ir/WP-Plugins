== Description ==
این افزونه امکان تحقیق کلمات کلیدی با استفاده از API چت‌جی‌پی‌تی را فراهم می‌کند. کاربر می‌تواند کلمه کلیدی اولیه را وارد کرده و افزونه لیستی از کلمات کلیدی مرتبط را بر اساس پاسخ API به او ارائه دهد.

== Installation ==
1. پوشه افزونه را به مسیر wp-content/plugins/ سایت وردپرس خود کپی کنید.
2. از بخش افزونه‌ها در وردپرس، افزونه را فعال کنید.
3. به منوی «Keyword Research» در بخش ادمین رفته و API Key مربوط به OpenAI را وارد کنید.
